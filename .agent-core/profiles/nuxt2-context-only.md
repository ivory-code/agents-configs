# Profile: nuxt2-context-only

> Context-only profile. 구현 규칙 강제 대상이 아니라 연계 플로우 파악용이다.

## Stack Snapshot

- Framework: Nuxt 2 + Vue 2
- Package manager: npm
- Path alias: `@/`
- Semicolon: `false`

## Commands

```bash
npm run lint
npm run build
npm run dev
```

## Notes

- React 구현 스킬(`code-style`, `design-system`, `testing`) 강제 적용 대상 아님
- 목적: 기존 플로우, 라우팅, 연동 영향도 파악
